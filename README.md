# 🚀 Planora – Smart Business Simulation Platform

## 📌 Overview
**Planora** is a smart web-based platform that helps entrepreneurs evaluate their business ideas before investing money.

The system allows users to input basic project data and receive:
- 📊 Financial analysis  
- 💰 Cash flow simulation  
- ⏳ Break-even estimation  
- 🤖 AI-powered insights  

---

## 🎯 Problem Statement
Many entrepreneurs start businesses without a clear financial plan, leading to high failure rates.

**Planora solves this by:**
- Simulating real-world financial scenarios  
- Helping users understand risks before launching  
- Supporting smarter decision-making  

---

## 💡 Key Features

### 🔹 Business Simulation
Users enter:
- Initial Investment  
- Expected Monthly Sales  
- Monthly Costs  
- Business Type  

➡️ The system generates:
- Monthly profit  
- Break-even point  
- Profit margin  

---

### 🔹 Advanced Scenario Modeling
Includes:
- Growth rate  
- Seasonality impact  
- Market stability  
- Marketing campaigns  
- Unexpected costs  

➡️ Produces more realistic simulations  

---

### 🔹 Interactive Dashboard
- 📈 Cash Flow Chart (monthly)  
- 🧾 Cost Breakdown (Donut Chart)  
- 📊 Scenario Comparison (Low / Average / High)  
- 📅 Monthly performance insights  

---

### 🔹 AI Insights
- Smart recommendations  
- Risk level detection  
- Strategic suggestions

## ⚙️ How It Works
1. User enters project data  
2. Data is sent to backend API  
3. Backend processes calculations  
4. Results are returned  
5. Dashboard displays insights  

---

## 📷 Demo Flow
1. Home Page → Introduction  
2. Simulation Page → Input data  
3. Dashboard → Results & charts  
4. AI Insights → Recommendations
